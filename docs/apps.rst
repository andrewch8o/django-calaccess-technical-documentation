Apps
====

This section dives into the full set of features available in each of our apps. It may only be appropriate for advanced users insterested using individual components of the CAL-ACCESS data pipeline.

.. toctree::
   :hidden:
   :maxdepth: 1

`django-calaccess-raw-data <raw_data_app.html>`_
    Downloads, extracts, cleans and loads the raw CAL-ACCESS data into a local database.

`django-calaccess-processed-data <processed_data_app.html>`_
    Transforms and refines the raw campaign finance data and lobbyist disclosure data into ready-to-analyze data models.
